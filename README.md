# 📊 Google Play Store Data Analysis (EDA & Feature Engineering)

## 📌 Objective
Analyze the Google Play Store dataset to uncover insights such as:
- Most popular app categories  
- Apps with highest installs  
- Top-rated applications  
- Installation and rating trends  

---

## 📁 Dataset Information
- 📊 Total Records: 10,841 rows  
- 📌 Features: 20 columns  
- 📂 Source: Google Play Store dataset  

---

## 🛠️ Tools & Technologies
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  

---

## 🔄 Project Workflow

### 1. Data Cleaning
- Removed invalid and duplicate records  
- Handled missing values  
- Converted data types:
  - Reviews → Integer  
  - Size → Float  
  - Installs & Price → Numeric  
- Processed "Last Updated" column into Day, Month, Year  

---

### 2. Exploratory Data Analysis (EDA)
- Analyzed numerical and categorical features  
- Performed univariate analysis  
- Visualized distributions using plots and graphs  
- Identified patterns in ratings, installs, and categories  

---

### 3. Feature Engineering
- Extracted new features from date column  
- Cleaned and standardized dataset for analysis  
- Prepared dataset for further insights  

---

## 📊 Key Insights
- 📌 **Family category** has the highest number of apps (~18%)  
- 🎮 **Game category** has the highest installs (~35 Billion installs)  
- ⭐ Identified **269 apps with 5-star ratings**  
- 📱 Most popular apps:
  - Games → Subway Surfers  
  - Communication → Hangouts  
  - Productivity → Google Drive  
  - Social → Instagram  

---

## 📈 Visualizations
- Distribution plots (Numerical features)  
- Count plots (Categorical features)  
- Pie chart (Category distribution)  
- Bar charts (Top categories & installs)
